    Add a new async function `embed_listing_chunk(listing_id: str) -> bool` to `backend/app/ai/embeddings.py`.

    This function:
    1. Fetches the listing from Supabase: select title, description, location, city, neighborhood_id, property_type, category, price, bedrooms, bathrooms, size_sqm, amenities, compound_name, status, deleted_at
    2. Returns False if deleted or not found
    3. Builds chunk_text in this exact format:
       ```
       {title}. {property_type} in {city}{, location if different}. {bedrooms} bed, {bathrooms} bath{, size_sqm sqm}. Price: {price} EGP. {description[:300] if description}. Amenities: {', '.join(amenities) if amenities}. {Compound: compound_name if compound_name}.
       ```
       Use only fields that are not None. Strip trailing commas and double spaces.
    4. Calls `await ollama.embed(chunk_text)` — returns False if empty list
    5. Upserts to knowledge_chunks using `on_conflict="source_type,source_id"` (Supabase upsert):
       ```python
       supabase_admin.table("knowledge_chunks").upsert({
           "source_type": "listing",
           "source_id": listing_id,
           "chunk_text": chunk_text,
           "embedding": vector,
           "metadata": {
               "city": listing.get("city", ""),
               "category": listing.get("category", ""),
               "price": float(listing.get("price", 0)),
               "bedrooms": listing.get("bedrooms"),
               "property_type": listing.get("property_type", ""),
           },
           "updated_at": "now()",
       }, on_conflict="source_type,source_id").execute()
       ```
    6. Returns True on success, False on any exception.

    Keep the existing `embed_listing()` function unchanged — it writes to listings.embedding and is still used for the recommendations vector search.

    Also add `delete_listing_chunk(listing_id: str) -> bool`:
    ```python
    async def delete_listing_chunk(listing_id: str) -> bool:
        try:
            supabase_admin.table("knowledge_chunks") \
                .delete() \
                .eq("source_type", "listing") \
                .eq("source_id", listing_id) \
                .execute()
            return True
        except Exception:
            return False
    ```
